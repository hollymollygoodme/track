# Track Project
Useful tool to track location or mobile number.

## Fitur Utama
* [1] IP Tracker
* [2] Phone Tracker
* [3] Username Tracker

## Cara Instalasi
Gunakan perintah berikut di terminal/Termux:

```bash
pkg install git
pkg install python3
git clone https://github.com/hollymollygoodme/track.git
cd track
pip3 install -r requirements.txt
python3 track.py
